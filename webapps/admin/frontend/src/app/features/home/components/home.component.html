Welcome

This webapp serves as a template webapp for the monorepo.
Its full stack, database keeps track of the total clicks.
It also has ratelimiting.

Ping here

Tech stack

  • Angular
  • Spring Boot
  • Kotlin
  • Docker + Docker compose

endpoints

base address: template.patrickdd.com/api/v1

/ping

sends ping to server and increments total clicks.

/ping/stats

shows data about pings including total click count.

/status

shows data about the server
